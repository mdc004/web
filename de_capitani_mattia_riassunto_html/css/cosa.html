Cosa sono i CSS

CSS sta per Cascading Style Sheets, in italiano fogli di stile. I css servono appunto per dare uno stile alla nostra pagina, a seconda di dove li posizioniamo assemono un nome diverso:

In linea, ovvero quelli visti nelle precedenti lezioni in html, che si trovano all'interno del tag da modificare

Interni, ovvero quelli che si trovano sempre all'interno della pagina html, ma fra i tag "style", nell' "head"

Esterni, che sono quelli che si trovano in un file esterno con estensione ".css" e che permettono di fare un solo file css per più pagine
