Come scrivere css interni ed esterni

I CSS esterni ed interni hanno la stessa sintassi, cambia solo la posizione di questi, vediamo ora come inserirli nel nostro sito:

Per i CSS interni ci basterà recarci nell' "head" e aprire e chiudere il tag "style", all interno di questo andremo a scrivere il nome della categoria di tag a cui vogliamo fare una modifica, separando un tag dall'altro con la virgola, infine non ci resta che aprire e chiudere le graffe e scrivere le stessa cose charset avremmo scritto nel css in linea, quindi: attributo:valore;

<*html> <*head> <*style> tag { attributo: valore; } <*/style> <*/head> <*body><*/body> <*/html>

La sintassi per i css esterni è uguale identica, ma senza il tag "style", tuttavia il procedimento per crearli e lincarli alla pagina è un pochino più completto, però ci mermetterà di ottenere un lavoro di gran lunga più ordinato.
Procediamo quindi creando un nuovo file, ma con estensione ".css", dopodiché ci dobbiamo recare nell' "head" del documento al quale vogliamo collegare i Css e scriviamo il seguente tag:

<*link rel="stylesheet" type="text/css" href="percorso/nome_file.css">

Fatto ciò potremo recarci nel nostro file con estensione ".css" e inizare a cambiare lo stile del nostro sito.

tag { attributo: valore; }

I CSS, quindi, modificano lo stile di un tag o di un gruppo di tag, purchè dello stesso tipo, ma nel caso in cui noi volessimo modificare il colore di due soli paragrafi e non di tutti quelli della pagina come dovremmo fare?
Per questo ci vengono in aiuto classi e id, infatti se andremo a inserire l'attributo "class" o l'attributo "id" potremo andare a modificare, all'interno dei css solo i tag che sono hanno un "id" o una "class" di un determinato tipo; la sintassi e la seguente:

Nell' HTML: <*tag class="nome_classe"> Nel CSS: .nome_classe { attributo: valore; } Nel' HTML: <*tag id="nome_id"> Nel CSS: #nome_id { attributo: valore; }
